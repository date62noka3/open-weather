# open-weather

3日分の天気情報を表示するアプリです。  
(取得できない時間帯があります。)  

# DEMO
![open_weather](https://user-images.githubusercontent.com/61310801/87107131-ef5f3380-c299-11ea-8811-6f9f10f4b5fc.gif)

# Description
【使用したライブラリ】  
Retrofit, Moshi, Coroutine, fuel  

【開発経緯】  
ポートフォリオ(名刺図鑑)はKotlinを触り始めてちょっとしたときに開発したものです。  
よいITソリューションになったかなとは感じていますが、  
ローカルで完結させてしまって忍びないので  
勉強の一環としてAPIを叩くアプリを開発しました。
